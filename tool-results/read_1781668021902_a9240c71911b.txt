    58→    58→## Overview
    59→    59→
    60→    60→**TheViralFindsMY** is a comprehensive, production-ready Shopee Affiliate management platform built specifically for the Malaysian market. It combines real-time product discovery, AI-powered content creation, intelligent automation, and deep analytics into a single, unified dashboard.
    61→    61→
    62→    62→The platform is designed to help Malaysian affiliates maximize their earnings through:
    63→    63→
    64→    64→- **Smart product discovery** with AI scoring and Commission XTRA detection
    65→    65→- **Automated content creation** in Manglish, Bahasa Malaysia, and English
    66→    66→- **Social media scheduling** optimized for Malaysian peak engagement times
    67→    67→- **Trend monitoring** with competitor intelligence and keyword tracking
    68→    68→- **HERMES AI Agent** for conversational insights, task automation, and skill management
    69→    69→- **Multi-platform support** — Shopee, TikTok Shop, Lazada in one dashboard (roadmap)
    70→    70→
    71→    71→> **Target Market:** Malaysia (RM currency, Shopee Malaysia, Malaysian consumer behavior)
    72→    72→> **Language Support:** English, Bahasa Malaysia, Manglish (Malaysian English)

   168→   168→### Design Principles
   169→   169→
   170→   170→1. **Dual Data Source** — Every Shopee API call gracefully falls back to realistic mock data when API access is unavailable
   171→   171→2. **AI-First** — All content generation, analysis, and insights use z-ai-web-dev-sdk with algorithmic fallbacks
   172→   172→3. **Demo-Ready** — The app is fully functional with demo data out of the box — no external accounts needed
   173→   173→4. **Malaysian-Optimized** — Currency, language, peak times, seasonal events, and consumer behavior tuned for Malaysia
   174→   174→5. **Client-Side Routing** — Single-page app using Zustand store for navigation (not Next.js file-based routing)

   231→   231→
   232→   232→### Installation
   233→   233→
   234→   234→```bash
   235→   235→# Clone the repository
   236→   236→git clone https://github.com/thisisniagahub/theviralfindsmy.git
   237→   237→cd theviralfindsmy
   238→   238→
   239→   239→# Install dependencies
   240→   240→bun install
   241→   241→
   242→   242→# Set up the database
   243→   243→bun run db:push
   244→   244→
   245→   245→# Start the development server
   246→   246→bun run dev
   247→   247→```

   250→   250→
   251→   251→### Environment Variables
   252→   252→
   253→   253→Create a `.env` file in the project root:
   254→   254→
   255→   255→```env
   256→   256→# Database
   257→   257→DATABASE_URL="file:./db/custom.db"
   258→   258→
   259→   259→# (Optional) Shopee Affiliate API
   260→   260→# SHOPEE_APP_ID=your_app_id
   261→   261→# SHOPEE_SECRET=your_secret

   266→   266→# NEXTAUTH_URL=http://localhost:3000

   365→   365→### 1. Dashboard & Analytics
   366→   366→
   367→   367→**Dashboard** (`/` → `dashboard` page)
   368→   368→
   369→   369→The central command center displaying real-time performance metrics:
   370→   370→
   371→   371→- **Stats Cards:** Total clicks, conversions, earnings, conversion rate with 30-day trend comparison
   372→   372→- **Activity Feed:** Recent clicks, conversions, and link activity
   373→   373→- **Top Products:** Best-performing affiliate products by earnings
   374→   374→- **Quick Actions:** Generate link, create campaign, post to social
   375→   375→
   376→   376→**Analytics** (`analytics` page)
   377→   377→
   378→   378→Deep-dive performance analysis:
   379→   379→
   380→   380→- **Time-series Charts:** Clicks and conversions over 7/30/90 days
   381→   381→- **Geographic Breakdown:** Click origins by country
   382→   382→- **Device Analysis:** Desktop vs mobile vs tablet distribution

   686→   686→### Shopee API Integration
   687→   687→
   688→   688→### Dual Data Source System
   689→   689→
   690→   690→The app implements an intelligent dual data source that automatically switches between real and mock data:
   691→   691→
   692→   692→```
   693→   693→┌──────────────┐     Credentials     ┌──────────────┐
   694→   694→│  User Input  │ ──────────────────► │  AppSetting  │
   695→   695→│  (Settings)  │                     │  (Database)  │
   696→   696→└──────────────┘                     └──────┬───────┘
   697→   697→                                            │
   698→   698→                              ┌──────────────▼──────────────┐
   699→   699→                              │    ShopeeAffiliateService    │
   700→   700→                              │                              │
   701→   701→                              │  if credentials valid:       │
   702→   702→                              │    → ShopeeGraphQLClient     │
   703→   703→                              │  else:                       │
   704→   704→                              │    → ShopeeMockService       │
   705→   705→                              └──────────────┬───────────────┘
   706→   706→                                             │
   707→   707→                              ┌──────────────▼───────────────┐
   708→   708→                              │   Unified Response Format    │
   709→   709→                              │   source: "graphql_api" |    │
   710→   710→                              │            "mock"            │
   711→   711→                              └──────────────────────────────┘
   712→   712→```
   713→   713→
   714→   714→Every API response includes a `source` field indicating whether data came from the real API (`graphql_api`) or mock service (`mock`).
   715→   715→
   716→   716→### GraphQL Client
   717→   717→
   718→   718→The `ShopeeGraphQLClient` implements the official Shopee Affiliate Open API:
   719→   719→
   720→   720→- **Authentication:** HMAC-SHA256 signature on every request
   721→   721→- **Regional Endpoints:** Malaysia, Singapore, Indonesia, Thailand, Philippines, Vietnam
   722→   722→- **Operations:** Product search, link generation, commission orders, click stats
   723→   723→- **Error Handling:** Graceful fallback to mock data on API errors
   724→   724→
   725→   725→### Webhook Events
   726→   726→
   727→   727→| Event | Description | Processing |
   728→   728→|-------|-------------|------------|

   731→   731→| `commission_update` | Commission status changed | Update conversion status |
