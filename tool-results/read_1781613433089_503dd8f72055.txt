    55→## Overview
    56→
    57→**TheViralFindsMY** is a comprehensive, production-ready Shopee Affiliate management platform built specifically for the Malaysian market. It combines real-time product discovery, AI-powered content creation, intelligent automation, and deep analytics into a single, unified dashboard.
    58→
    59→The platform is designed to help Malaysian Shopee affiliates maximize their earnings through:
    60→
    61→- **Smart product discovery** with AI scoring and Commission XTRA detection
    62→- **Automated content creation** in Manglish, Bahasa Malaysia, and English
    63→- **Social media scheduling** optimized for Malaysian peak engagement times
    64→- **Trend monitoring** with competitor intelligence and keyword tracking
    65→- **HERMES AI Agent** for conversational insights, task automation, and skill management
    66→
    67→> **Target Market:** Malaysia (RM currency, Shopee Malaysia, Malaysian consumer behavior)
    68→> **Language Support:** English, Bahasa Malaysia, Manglish (Malaysian English)

   138→### Design Principles
   139→
   140→1. **Dual Data Source** — Every Shopee API call gracefully falls back to realistic mock data when API access is unavailable
   141→2. **AI-First** — All content generation, analysis, and insights use z-ai-web-dev-sdk with algorithmic fallbacks
   142→3. **Demo-Ready** — The app is fully functional with demo data out of the box — no external accounts needed
   143→4. **Malaysian-Optimized** — Currency, language, peak times, seasonal events, and consumer behavior tuned for Malaysia
   144→5. **Client-Side Routing** — Single-page app using Zustand store for navigation (not Next.js file-based routing)

   194→## Getting Started
   195→
   196→### Prerequisites
   197→
   198→- **Node.js** >= 18
   199→- **Bun** (recommended) or npm
   200→- **Git**
   201→
   202→### Installation
   203→
   204→```bash
   205→# Clone the repository
   206→git clone https://github.com/thisisniagahub/theviralfindsmy.git
   207→cd theviralfindsmy
   208→
   209→# Install dependencies
   210→bun install
   211→
   212→# Set up the database
   213→bun run db:push
   214→
   215→# Start the development server
   216→bun run dev
   217→```
   218→
   219→The app will be available at `http://localhost:3000`.
   220→
   221→### Environment Variables
   222→
   223→Create a `.env` file in the project root:
   224→
   225→```env
   226→# Database
   227→DATABASE_URL="file:./db/custom.db"
   228→
   229→# (Optional) Shopee Affiliate API
   230→# SHOPEE_APP_ID=your_app_id
   231→# SHOPEE_SECRET=your_secret
   232→# SHOPEE_REGION=my
   233→
   234→# (Optional) NextAuth
   235→# NEXTAUTH_SECRET=your_secret
   236→# NEXTAUTH_URL=http://localhost:3000
   237→```
   238→
   239→> **Note:** The app works fully in demo mode without any API credentials. All Shopee features use realistic Malaysian market mock data.
   240→
   241→### Available Scripts
   242→
   243→| Script | Description |
   244→|--------|-------------|
   245→| `bun run dev` | Start development server on port 3000 |
   246→| `bun run build` | Build for production |
   247→| `bun run lint` | Run ESLint checks |
   248→| `bun run db:push` | Push Prisma schema to database |
   249→| `bun run db:generate` | Generate Prisma client |

   654→## Shopee API Integration
   655→
   656→### Dual Data Source System
   657→
   658→The app implements an intelligent dual data source that automatically switches between real and mock data:
   659→
   660→```
   661→┌──────────────┐     Credentials     ┌──────────────┐
   662→│  User Input  │ ──────────────────► │  AppSetting  │
   663→│  (Settings)  │                     │  (Database)  │
   664→└──────────────┘                     └──────┬───────┘
   665→                                            │
   666→                              ┌──────────────▼──────────────┐
   667→                              │    ShopeeAffiliateService    │
   668→                              │                              │
   669→                              │  if credentials valid:       │
   670→                              │    → ShopeeGraphQLClient     │
   671→                              │  else:                       │
   672→                              │    → ShopeeMockService       │
   673→                              └──────────────┬───────────────┘
   674→                                             │
   675→                              ┌──────────────▼───────────────┐
   676→                              │   Unified Response Format    │
   677→                              │   source: "graphql_api" |    │
   678→                              │            "mock"            │
   679→                              └──────────────────────────────┘
   680→```
   681→
   682→Every API response includes a `source` field indicating whether data came from the real API (`graphql_api`) or mock service (`mock`).
   683→
   684→### GraphQL Client
   685→
   686→The `ShopeeGraphQLClient` implements the official Shopee Affiliate Open API:
   687→
   688→- **Authentication:** HMAC-SHA256 signature on every request
   689→- **Regional Endpoints:** Malaysia, Singapore, Indonesia, Thailand, Philippines, Vietnam
   690→- **Operations:** Product search, link generation, commission orders, click stats
   691→- **Error Handling:** Graceful fallback to mock data on API errors
   692→
   693→### Webhook Events
   694→
   695→| Event | Description | Processing |
   696→|-------|-------------|------------|
   697→| `order_conversion` | Customer completed purchase | Save conversion + create notification |
   698→| `click` | Affiliate link clicked | Increment click counter |
   699→| `commission_update` | Commission status changed | Update conversion status |
   700→
   701→Webhook verification uses HMAC-SHA256 signature validation.
   702→
   703→### Getting Shopee API Access
   704→
   705→1. Register at [Shopee Affiliate Program](https://affiliate.shopee.com.my/)
   706→2. Go to **Account > API > Open API**
   707→3. Request access to the Shopee Affiliate Open API Platform
   708→4. Generate your **App ID** and **Secret**
   709→5. Enter credentials in **Settings > Shopee Integration**
   710→
   711→> **Important:** Shopee Affiliate Open API uses GraphQL specification. You must request access separately — having an App ID/Secret is not sufficient without API platform access.
