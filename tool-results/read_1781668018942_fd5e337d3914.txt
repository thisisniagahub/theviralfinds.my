    58→## Overview
    59→
    60→**TheViralFindsMY** is a comprehensive, production-ready Shopee Affiliate management platform built specifically for the Malaysian market. It combines real-time product discovery, AI-powered content creation, intelligent automation, and deep analytics into a single, unified dashboard.
    61→
    62→The platform is designed to help Malaysian affiliates maximize their earnings through:
    63→
    64→- **Smart product discovery** with AI scoring and Commission XTRA detection
    65→- **Automated content creation** in Manglish, Bahasa Malaysia, and English
    66→- **Social media scheduling** optimized for Malaysian peak engagement times
    67→- **Trend monitoring** with competitor intelligence and keyword tracking
    68→- **HERMES AI Agent** for conversational insights, task automation, and skill management
    69→- **Multi-platform support** — Shopee, TikTok Shop, Lazada in one dashboard (roadmap)
    70→
    71→> **Target Market:** Malaysia (RM currency, Shopee Malaysia, Malaysian consumer behavior)
    72→> **Language Support:** English, Bahasa Malaysia, Manglish (Malaysian English)

   168→### Design Principles
   169→
   170→1. **Dual Data Source** — Every Shopee API call gracefully falls back to realistic mock data when API access is unavailable
   171→2. **AI-First** — All content generation, analysis, and insights use z-ai-web-dev-sdk with algorithmic fallbacks
   172→3. **Demo-Ready** — The app is fully functional with demo data out of the box — no external accounts needed
   173→4. **Malaysian-Optimized** — Currency, language, peak times, seasonal events, and consumer behavior tuned for Malaysia
   174→5. **Client-Side Routing** — Single-page app using Zustand store for navigation (not Next.js file-based routing)

   224→## Getting Started
   225→
   226→### Prerequisites
   227→
   228→- **Node.js** >= 18
   229→- **Bun** (recommended) or npm
   230→- **Git**
   231→
   232→### Installation
   233→
   234→```bash
   235→# Clone the repository
   236→git clone https://github.com/thisisniagahub/theviralfindsmy.git
   237→cd theviralfindsmy
   238→
   239→# Install dependencies
   240→bun install
   241→
   242→# Set up the database
   243→bun run db:push
   244→
   245→# Start the development server
   246→bun run dev
   247→```
   248→
   249→The app will be available at `http://localhost:3000`.
   250→
   251→### Environment Variables
   252→
   253→Create a `.env` file in the project root:
   254→
   255→```env
   256→# Database
   257→DATABASE_URL="file:./db/custom.db"
   258→
   259→# (Optional) Shopee Affiliate API
   260→# SHOPEE_APP_ID=your_app_id
   261→# SHOPEE_SECRET=your_secret
   262→# SHOPEE_REGION=my
   263→
   264→# (Optional) NextAuth
   265→# NEXTAUTH_SECRET=your_secret
   266→# NEXTAUTH_URL=http://localhost:3000
   267→```
   268→
   269→> **Note:** The app works fully in demo mode without any API credentials. All Shopee features use realistic Malaysian market mock data.
   270→
   271→### Available Scripts
   272→
   273→| Script | Description |
   274→|--------|-------------|
   275→| `bun run dev` | Start development server on port 3000 |
   276→| `bun run build` | Build for production |

   685→
   686→### Shopee API Integration
   687→
   688→### Dual Data Source System
   689→
   690→The app implements an intelligent dual data source that automatically switches between real and mock data:
   691→
   692→```
   693→┌──────────────┐     Credentials     ┌──────────────┐
   694→│  User Input  │ ──────────────────► │  AppSetting  │
   695→│  (Settings)  │                     │  (Database)  │
   696→└──────────────┘                     └──────┬───────┘
   697→                                            │
   698→                              ┌──────────────▼──────────────┐
   699→                              │    ShopeeAffiliateService    │
   700→                              │                              │
   701→                              │  if credentials valid:       │
   702→                              │    → ShopeeGraphQLClient     │
   703→                              │  else:                       │
   704→                              │    → ShopeeMockService       │
   705→                              └──────────────┬───────────────┘
   706→                                             │
   707→                              ┌──────────────▼───────────────┐
   708→                              │   Unified Response Format    │
   709→                              │   source: "graphql_api" |    │
   710→                              │            "mock"            │
   711→                              └──────────────────────────────┘
   712→```
   713→
   714→Every API response includes a `source` field indicating whether data came from the real API (`graphql_api`) or mock service (`mock`).
   715→
   716→### GraphQL Client
   717→
   718→The `ShopeeGraphQLClient` implements the official Shopee Affiliate Open API:
   719→
   720→- **Authentication:** HMAC-SHA256 signature on every request
   721→- **Regional Endpoints:** Malaysia, Singapore, Indonesia, Thailand, Philippines, Vietnam
   722→- **Operations:** Product search, link generation, commission orders, click stats
   723→- **Error Handling:** Graceful fallback to mock data on API errors
   724→
   725→### Webhook Events
   726→
   727→| Event | Description | Processing |
   728→|-------|-------------|------------|
   729→| `order_conversion` | Customer completed purchase | Save conversion + create notification |
   730→| `click` | Affiliate link clicked | Increment click counter |
   731→| `commission_update` | Commission status changed | Update conversion status |
   732→
   733→Webhook verification uses HMAC-SHA256 signature validation.
   734→
   735→### Getting Shopee API Access
   736→
   737→1. Register at [Shopee Affiliate Program](https://affiliate.shopee.com.my/)
   738→2. Go to **Account > API > Open API**
   739→3. Request access to the Shopee Affiliate Open API Platform
   740→4. Generate your **App ID** and **Secret**
   741→5. Enter credentials in **Settings > Shopee Integration**
   742→
   743→> **Important:** Shopee Affiliate Open API uses GraphQL specification. You must request access separately — having an App ID/Secret is not sufficient without API platform access.
